## Comment lire la série

1. Commencer par le **volet 1** (Andreessen) pour le texte déclencheur.
2. **Volet 2** (Thiel) pour le versant politique/anti-démocratique.
3. **Volets 3 & 4** pour les racines intellectuelles et les lignes de faille.
4. **Volet 5 + carte HTML** pour la vue d'ensemble et la prospective.
